# Gym Coaching Platform - Architecture Diagrams

This document presents four comprehensive Mermaid diagrams that illustrate the architecture of the Gym Coaching Platform. Two approaches are contrasted throughout:

- **Approach A**: Modular monolith with async queues (RabbitMQ)
- **Approach B**: Full microservices with event-driven architecture (Kafka)
